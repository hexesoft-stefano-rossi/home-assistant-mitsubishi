# Hexesoft Mitsubishi Bridge Home Assistant

Questo programma è il "cervello" che mette in comunicazione il tuo sistema
di climatizzazione **Mitsubishi** (tramite i controller AE-200, EW-50 o AG-150A) 
con **Home Assistant**.

This program is the "brain" that connects your **Mitsubishi** HVAC system 
(via AE-200, EW-50, or AG-150A controllers) with **Home Assistant**.
